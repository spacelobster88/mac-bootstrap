The Name

Why Auspex

Before Rome's legions marched, the auspex read the signs and declared the ground fit for action. Auspex does the same for your Mac: it reads the machine, installs every dependency, and prepares the ground so your AI services can march in formation from the first command.

Install Dependencies

Installs Homebrew, Python, Go, Node.js, Ollama, Claude CLI, and GitHub CLI. Grant Terminal accessibility and full disk access first.

git clone https://github.com/spacelobster88/auspex.git cd auspex && chmod +x *.sh ./install.sh
2
Clone, Configure & Start

Clones all service repos, builds projects, prompts for secrets (API keys, bot tokens), and installs LaunchAgent services.

gh auth login && claude login ./setup.sh
3
Verify

Runs health checks against all six services to confirm they are up and responding.

./health-check.sh
Versioning

Reproducible deployments

Auspex pins service versions in stack.json for reproducible deployments. setup.sh checks out exact pinned commits rather than pulling main. Commit the updated stack.json to deploy. Revert to roll back.

# Bump a service to a specific commit ./scripts/bump-version.sh mini-claude-bot abc1234def5678 # Bump to current main HEAD ./scripts/bump-version.sh mini-claude-bot
Capabilities

Features

Everything you need to run a production AI agent hub on a single Mac.

Provisioning
One-Command Setup

Three shell scripts take a bare Mac from zero to a fully operational AI hub. Homebrew, Python, Go, Node.js, Ollama, and Claude CLI — all automated.

Orchestration
LaunchAgent Services

All six services run as macOS LaunchAgents with KeepAlive and RunAtLoad. Auto-restart on failure, boot on login, zero manual intervention.

Versioning
Stack Snapshots

Pin all service versions in stack.json. Take coordinated snapshots across all repos. Roll back the entire stack to any previous state with one command.

Communication
Telegram Remote Control

Control your AI agent hub from anywhere via Telegram. Send commands, receive notifications, monitor status — all from your phone.

Intelligence
Semantic Memory

Vector-based semantic search over all chat history using Ollama embeddings. Context-aware responses that remember past conversations and decisions.

Safety
Hardware-Aware Scheduling

Centurion monitors RAM, CPU, and memory pressure in real time. Admission control prevents OOM kills. K8s-style scheduling for agent fleets.

Autonomy
Meta-Cognition Loop

AROS meta-loop evaluates system performance against defined goals, derives improvement signals, and autonomously evolves the stack over time.

Development
Project Orchestrator

Harness-loop decomposes projects into task DAGs, executes phases in parallel via subagents, with structured QA gates and cross-session resume.

Operations
Health Monitoring

Built-in health checks verify all services are up and responding. Automated cron jobs for daily reports, email triage, and system maintenance.
